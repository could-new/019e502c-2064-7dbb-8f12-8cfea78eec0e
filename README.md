# JNNCE Shivamogga Official Website

This is a cross-platform Flutter application serving as the official web portal for Jawaharlal Nehru New College of Engineering (JNNCE), Shivamogga. 

## Features
*   **Responsive Design:** Fully responsive layout working on Mobile, Tablet, Desktop, and Web.
*   **Home Screen:** Showcases the hero section, announcements, and quick highlights.
*   **About JNNCE:** Detailed history, vision, and mission of the institution.
*   **Academics:** Comprehensive list of engineering and management departments.

## Tech Stack
*   **Framework:** Flutter
*   **Language:** Dart
*   **Platforms:** Web, iOS, Android, macOS, Windows, Linux

## Setup & Run
1. Install [Flutter SDK](https://flutter.dev/docs/get-started/install)
2. Clone this repository
3. Run `flutter pub get`
4. Run `flutter run -d chrome` to test the web view locally.
